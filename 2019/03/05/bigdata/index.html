网络配置

1.打开 VMware -> 编辑 -> 虚拟网络编辑器
2.将 VMware 当成一个路由器使用，设置路由器网段 192.168.59.0

设置网络方式
1.打开 Linux 中 System -> Preferences -> Network Connections
2.使用图形化界面操作将 ip 设置成 192.168.59.250
